\# Portfolio Optimization Under Estimation Uncertainty



This project studies how estimation uncertainty affects mean-variance portfolio optimization using a small ETF portfolio. It compares a standard plug-in optimizer with Bayesian and bootstrap-based approaches.



\## Project structure



\- `data/raw/` - manually downloaded ETF price data

\- `notebooks/` - main analysis notebook



\## Main methods



\- Mean-variance optimization

\- Bayesian inference

\- MCMC

\- Bootstrapping
